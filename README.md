# Traverse: HexTek

### A magic and tech exploration modpack for NeoForge 1.21.1 | Alpha Development Build

## Download
[Download Latest Release](https://www.technicpack.net/modpack/traverse-official.2016393)

## About
Traverse: HexTek is a NeoForge 1.21.1 modpack built around the fusion of deep magic systems and complex tech automation, set against a world worth exploring. Five distinct magic systems — anchored by Ars Nouveau's spell-crafting ecosystem — run alongside four tech systems including AE2, Mekanism, Create, and Allthemodium progression. Seven custom dimensions, a fully rebuilt worldgen stack using the Stardust Labs suite, and over a dozen structure mods give the world density and purpose. Every system is designed to intersect — magic bridges to tech, tech feeds back into magic — so no playstyle exists in isolation.
This is an alpha development build. Not for public distribution.

## What is Traverse?
The world should feel worth walking through. That's the design principle behind Traverse: HexTek. Terralith reshapes the overworld with 85+ biomes. Nullscape rebuilds the End. Incendium rebuilds the Nether. YUNG's Suite replaces every vanilla structure with something worth finding. Moog's Suite fills the skies, seas, and hell with new places to stumble into. Seven dimensions — from the Aether to the Twilight Forest to the Undergarden — give the world vertical depth beyond the overworld.

Traverse is not a pack you rush through to reach endgame automation. The world is the content.

## Player Types
The Arcanist — Your path starts with Ars Nouveau and never really ends. Five magic systems, 34 magic mods, and a spell crafting ecosystem that bridges into AE2, Create, and Mekanism mean your magic tools never hit a ceiling.

The Engineer — AE2 networks, Mekanism ore processing, Create automation, and Draconic Evolution endgame give you a full tech progression from early RF generation to ME-driven megabuilds.

The Explorer — Seven dimensions, six structure categories, YUNG's full vanilla overhaul suite, and a Stardust Labs worldgen stack built for discovery. There is always somewhere new to go.

The Builder — 25 building mods including the full Macaw's suite, Chipped, Domum Ornamentum, and connected glass systems give you the blocks to make the things you build actually look like you meant them.

The Hybrid — The pack is designed for this. Magic and tech are bridged at the system level. Ars bridges to Create, AE2, and Mekanism. Soulshards feeds into AE2. Mystical Agriculture integrates with automation. Playing one system in isolation is a choice, not a requirement.

## Installation
Traverse: HexTek is currently in alpha development and is available through any public launcher. Playtesters wanted.

Download the latest release from the Releases page — look for Traverse-HexTek-v0.1.0-Alpha

Install NeoForge 21.1.216 for Minecraft 1.21.1

Drop the mods folder into your NeoForge instance

Allocate a minimum of 8GB RAM — 12GB recommended

Run Chunky to pre-generate your world before inviting other players to the server

Do not redistribute this build. It is an active development snapshot and is not representative of a final release.

Or simply look up the modpack in the technic launcher.
